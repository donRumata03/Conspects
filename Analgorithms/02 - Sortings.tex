Инвариант - поле i-го шага отсортирован префикс {[}0, i{]}

\begin{verbatim}
for i=0..n-1:
    j = i
    while j > 0 && a[j] < a[j - 1]:
        swap(a[j], a[j - 1])
        j--
\end{verbatim}

Время работы: \(n^2\)
